# Caltech Library - Thesis Embargo
Private repo for SPD 2.1 - Industry Collab

# Activating Pipenv
- Install pipenv:  `brew install pipenv`
- [On first run only]: `pipenv shell`
- To install packages: `pipenv install google-api-python-client google-auth-httplib2 google-auth-oauthlib python-dotenv`

# Edit .envSAMPLE
- Rename `.envSAMPLE` to `.env`
- Insert keys/ID
