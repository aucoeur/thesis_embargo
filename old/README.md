# Activating Pipenv
- Install pipenv:  `brew install pipenv`
- [On first run only]: `pipenv shell`
- To install packages: `pipenv install google-api-python-client google-auth-httplib2 google-auth-oauthlib python-dotenv gspread oauth2client`
